Introduction

This Page will show you how to install Drag-Move-Drop patch into a GWT library build. The purpose of DmD is to be integrated on GWT ; it is for this reason that the philosophy's project is to be (normally) only usable by a patch (as if I want many people discover it by package too in order to test it).

Linux Install Guide :

The first thing to do is to download the patch. Then create a folder, for example, dmd.

  • Please make sure that you have a subversion client : http://subversion.tigris.org/

Then, you'll have to checkout the source from gwt, follow this instructions :

  • Checkout or download the tools into the dmd folder :

svn checkout http://google-web-toolkit.googlecode.com/svn/tools/ tools

  • Checkout or download the trunk into the dmd folder :

svn checkout http://google-web-toolkit.googlecode.com/svn/trunk/ trunk

  • go into trunk folder
  • copy the Drag Drop patch into it
  • Apply the patch :

patch -p0 < ./GWT-DragMoveDrop-V2.patch

  • Please make sure that you have a ant command : http://ant.apache.org/bindownload.cgi
  • build a GWT project tipping :

ant

When finish, go into trunk/build/dist directory and use the corresponding file to your operating system and use the source as if it was the last version of GWT :)

If you have any trouble to use ant command, try to tip this command :

export GWT_TOOLS=~/dmd/tools

else the complete procedure is explain on this page : http://code.google.com/webtoolkit/makinggwtbetter.html#compiling

Windows Install Guide

The first thing to do is to download the patch. Then create a folder, for example, dmd.

  • Please make sure that you have a subversion client : http://subversion.tigris.org/

Then, you'll have to checkout the source from gwt, follow this instructions :

  • Checkout or download the tools into the dmd folder :

svn checkout http://google-web-toolkit.googlecode.com/svn/tools/ tools

  • Checkout or download the trunk into the dmd folder :

svn checkout http://google-web-toolkit.googlecode.com/svn/trunk/ trunk

  • go into trunk folder
  • copy the Drag Drop patch into it
  • You'll need to install a the TortoiseSVN sofware, so that you can patch the project : http://tortoisesvn.tigris.org/
  • type :

mkdir user\src\com\google\gwt\user\client\dnd & cd user\src\com\google\gwt\user\client\dnd & type nul>"DragListenerCollection.java" & type nul>"DragDrop.gwt.xml" & type nul>"DragListener.java" & type nul>"DragListenerAdapter.java" & type nul>"DropListenerCollection.java" & type nul>"SourcesDragEvents.java" & type nul>"DropListener.java" & type nul>"DropListenerAdapter.java" & type nul>"MouseDragGestureRecognizer.java" & type nul>"SourcesDropEvents.java"

  • open a work panel, go into your dmd folder
  • right click on the trunk folder, then click on TurtoiseSVN > Apply Patch
  • select the GWT-DragMoveDrop-V2.patch
  • Right click on the File Patch List, then click on Patch all
  • Define your tools environnement variable :

set GWT_TOOLS=C:\dmd\tools

set JAVA_HOME=C:\Program Files\Java\jdkXXXXX

  • Return to your trunk folder (on the cmd console) so that you can build a GWT project
  • Please make sure that you have a ant command : http://ant.apache.org/bindownload.cgi
  • build a GWT project tipping :

ant

When finish, go into trunk/build/dist directory and use the corresponding file to your operating system and use the source as if it was the last version of GWT :)

The complete normal procedure is explain on this page : http://code.google.com/webtoolkit/makinggwtbetter.html#compiling
